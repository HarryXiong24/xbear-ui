# Icon

---

## API

属性说明如下：

| 属性    | 说明     | 类型         | 默认值 | 版本 |
| ------- | -------- | ------------ | ------ | ---- |
| `icon`  | 图标名称 | `string`     | none   |      |
| `theme` | 主题     | `ThemeProps` | none   |      |

---

## 类型声明

```ts
export type ThemeProps = 'primary' | 'secondary' | 'success' | 'info' | 'warning' | 'danger' | 'light' | 'dark';
```
